# MCCompanion Docs

Welcome to the MCCompanion documentation. Here you'll find everything you need to get connected and start playing.

---

## What is MCCompanion?

MCCompanion is an app that lets you join **any Minecraft server** from your console

Available on **Windows**, **macOS**, **Android**, and **iOS**.
