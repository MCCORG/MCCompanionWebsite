# Multiplayer Connection Failed

Getting a "Multiplayer Connection Failed" message? This one is actually **not a NetherLink issue**.

NetherLink did its job it successfully redirected you to the server. The error is coming from the server side, not the app.

---

## Why does this happen?

- The target server is **offline or overloaded**
- The server requires a **different client version** or a specific Minecraft edition
- There's a **network issue** between the NetherLink server and the target server

---

## What to do

**Try a different server**
If it works on another server, the original one is likely just down or having issues.

**Check if the server is online**
Most servers have a Discord or website where they post status updates. Worth a quick check before spending time troubleshooting.

**Contact the server's support**
If the server should be online but keeps failing, reach out to them directly it's on their end.

**Still failing on every server?**
That's unusual. Enable **Debug Mode** in NetherLink and check the logs if it's happening across multiple servers it might point to a NetherLink issue.
