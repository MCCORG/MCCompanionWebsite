# Nintendo DNS Not Working

If the DNS setup isn't working on your Switch, run through these checks.

---

## Checklist

**1. Did you send the DNS config from the app?**
In NetherLink, make sure you actually hit **"Start Nintendo"**.

**2. Is the NetherLink IP entered correctly on the Switch?**
Go back to **System Settings → Internet → your network → Change Settings → DNS** and double check the Primary DNS matches exactly what the app showed you. One wrong digit breaks it.

**3. Is NetherLink online?**
Try switching between EU and US to see if one is more responsive. If both seem unresponsive, there may be a temporary outage.

**4. Is your network blocking custom DNS?**
Some networks especially school or public wifi with captive portals don't allow custom DNS settings. If that's the case, try on a different network like your mobile hotspot.

---

## Still not working?

- Enable **Debug Mode** in NetherLink and check the logs for any specific errors
- As an alternative, try **Friends Mode** instead of DNS mode it works differently and might get you connected without the DNS setup

---

:::tip
After you're done playing, remember to set your Switch DNS back to **Automatic** so your normal internet works properly.
:::
