# Java Mode

Java Mode lets you join **Java Edition servers** from Minecraft Bedrock.

---

## Steps

**1. Select Java Mode in the app**

Open NetherLink and switch to Java mode.

**2. Enter the server address and port**

Type in the Java Edition server IP and port. The default port is `25565` if the server doesn't specify one.

**3. Press "Start Java Mode"**

NetherLink will bridge the connection between Bedrock and the Java server.

**4. Open Minecraft Bedrock and go to the Friends tab**

Look for a LAN world called **"NetherLink"** and select it to join.

**5. Login to you Microsoft account with a valid Minecraft Java Account**

When logging in a screen will pop up with a Microsoft link code. You need to log into you Java Minecraft Email!


---

## ⚠️ Important warnings

Please read these before using Java Mode:

- A valid **Java Edition account (Microsoft)** is required
- Some servers use **anti-cheat systems** that may detect and ban your account
- Certain servers **explicitly prohibit Bedrock clients** always check the server rules before joining
- NetherLink is **not responsible** for any account bans, suspensions, or other account-related issues that result from using this feature
- **Use at your own risk**
