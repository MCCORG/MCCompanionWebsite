# Java Mode

Java Mode lets you join **Java Edition servers** from Minecraft Bedrock.

---

## Steps

1. **Connect everything to the same network**

Make sure your phone/tablet running NetherLink and your console are on the **same Wi‑Fi** *(avoid Guest Wi‑Fi if possible)*.

2. **Open NetherLink**

Launch the app on your phone or tablet.

3. **Enter your server details**

Enter the **server address** *(IP or domain)* and **port** *(default: **25565**)*, or pick one from your saved/recommended servers.

4. **Select a server and select Java as the mode**

Tap on the server you prefer and select **Java**.

5. **Start Java**

Tap **Start Java**. NetherLink will send the configuration to its servers.

6. **Join from your console**

On your console, open **Minecraft → Play → Friends/worlds** and look for a **LAN Game** NetherLink. Tap it to connect.

7. **Log in to the Microsoft account that owns Minecraft Java Edition**

When logging in, a screen will pop up with a Microsoft link code. Use it to sign in to the Microsoft account that owns Minecraft Java Edition.

8. **Stop when you’re done**

When finished, return to NetherLink and tap **Stop** to end the broadcast.

---

## ⚠️ Important warnings

Please read these before using Java Mode:

- A valid **Java Edition account (Microsoft)** is required
- Packet translation may not be perfect. It could trigger the server’s anti-cheat system and result in a ban. We are not responsible in such cases
- Certain servers **explicitly prohibit Bedrock clients** always check the server rules before joining
- NetherLink is **not responsible** for any account bans, suspensions, or other account-related issues that result from using this feature
- **Use at your own risk**
