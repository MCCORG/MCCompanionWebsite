Nintendo Switch

Join a Minecraft server on your Nintendo Switch

Before you start​

Make sure:

  • NetherLink is open on your PC, Mac, or phone
  • Your Switch is connected to the same network
  • You know which NetherLink server you want to use (EU: 161.97.182.113 or US: 217.77.15.138)

Steps​

1. Open NetherLink and switch to Nintendo mode

Select your preferred server EU or US, pick the one closest to you.

2. Tap "Send DNS Config"

This pushes the DNS settings to the NetherLink server so it knows where to send your traffic.

3. On your Switch, go to DNS settings

Navigate to: System Settings → Internet → Internet Settings → (your network) → Change Settings → DNS

Set the Primary DNS to the NetherLink IP shown in the NetherLink app.

4. Open Minecraft and join a featured server

Your connection will be silently redirected to your server through the NetherLInk. No extra steps needed ingame.

Good to know​

  • This is not a LAN broadcast the Switch won't see a LAN world, traffic is routed at the DNS level
  • When you're done playing, revert your DNS back to automatic if you need your normal internet behavior
  • If the connection feels slow, try switching between the EU and US NetherLink servers to see which works better for you
tip

If Minecraft can't connect at all after changing the DNS, double check the NetherLInk IP was copied correctly — a single wrong digit will break it.
