# Friends Mode

Join a minecraft server on any console with the friend mode.
## Before you start

- Add the **NetherLink NetherLink friend account** on your console and make sure the friend request is accepted

---

## Steps

**1. Add the NetherLink friend account**

On your console, add the NetherLink account to your friends list. If there's a pending request, accept it first — Friends mode won't work without this.

**2. Enable Friends mode in NetherLink**

Open the app, switch to Friends mode and press start. NetherLink will tell you which friend to add.

**3. Open Minecraft and go to Friends**

Navigate to **Play → Friends tab** and look for LAN Worlds. Your server should show up there.

**4. Select it and join**

That's it you and your friends should all be able to connect.

---

## Not showing up?

- Try refreshing the Friends tab it doesn't always update on its own
- Restart Minecraft and enable Friends mode again before rejoining
- Double check the NetherLink friend account is accepted, not just added

---

:::tip
If your friends can see the world but can't join, it's usually a NAT issue on their side. Ask them to check their network settings or try a different connection.
:::
